# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability, **do not open a public issue**.

Send a private report via:
1. **GitHub Security Advisories** — navigate to the repo's Security tab and create a new advisory.
2. **Direct message** — contact the maintainer at [@ForgedEmir](https://github.com/ForgedEmir).

You should receive a response within 48 hours.

## What to Include

- Description of the vulnerability
- Steps to reproduce (PoC preferred)
- Potential impact

## Disclosure

We believe in coordinated disclosure. Please give us reasonable time to fix
the issue before publishing it publicly.
